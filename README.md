# Estudio Jurídico Ferreyra López

Web profesional para el estudio jurídico Ferreyra López.

## 📋 Descripción

Sitio web responsivo y moderno para un estudio jurídico, con:
- Diseño elegante con fondo azul degradado
- Botones de contacto directo (Llamada, Email, WhatsApp)
- Iconos de redes sociales
- Versión responsive para móviles
- Animaciones suaves

## 🚀 Características

- ✅ HTML semántico
- ✅ CSS responsivo
- ✅ JavaScript vanilla
- ✅ Animaciones fluidas
- ✅ Diseño mobile-first
- ✅ Fácil de mantener y personalizar

## 📁 Estructura

```
ferreyra-lopez-estudio/
├── index.html      # Página principal
├── styles.css      # Estilos
├── script.js       # Interactividad
├── assets/         # Recursos (logo, imágenes)
└── README.md       # Este archivo
```

## 🎨 Personalización

### Colores
Puedes cambiar los colores en `styles.css`:
- Fondo: `background: linear-gradient(135deg, #001a4d 0%, #003d99 50%, #001a4d 100%);`
- Botón de llamada: `.btn-phone` (color: #4a7c7e)
- Botón de email: `.btn-email` (color: #2196f3)
- Botón de WhatsApp: `.btn-whatsapp` (color: #25d366)

### Textos
Todos los textos están en `index.html` y son fáciles de editar.

## 📱 Responsive

El sitio está optimizado para:
- Desktop (1920px y más)
- Tablet (768px - 1919px)
- Mobile (480px - 767px)
- Mobile pequeño (menos de 480px)

## 🔗 Enlaces de redes sociales

Actualiza los enlaces en `index.html` en la sección de social icons.

## 📞 Funcionalidad de botones

Los botones están listos para configurar con:
- `tel:` para llamadas
- `mailto:` para emails
- Links de WhatsApp API
